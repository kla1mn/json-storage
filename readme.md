1) Установить python 3.14
2) Далее просто написать в консоли make init
3) make start - запуск приложения


uv - аналог pip, но более быстрый и удобный

краткий гайд:
uv add ... - добавить библиотеку ... в проект
uv sync - подтягивает все версии библиотек указанных в pyproject-toml
uv sync --all-extras это если нужны зависимости dev (отдельная группа библиотек, нужно для тестов)
